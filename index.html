Appointment Booking System :
This Appointment Booking System provides a convenient and efficient method for booking appointments online.
Built using the MERN stack (MongoDB, Express.js, React.js, and Node.js), the system allows users to register,
view available time slots, book or cancel appointments, and make secure payments. Admins can manage doctors,
appointments, and time slots through a dashboard. The system enhances accessibility, reduces dependency
on manual systems, and ensures smooth interaction between users and service providers.
